如何使用（其他编程语言相类似）
========================

0: 需要到这里注册 http://chongdata.com/old/register.php 获取帐号密码 (username和password)

PHP
---
1: 需要安装php-curl，譬如在linux:

$ yum install php-curl

2: cd进入对应的目录用php运行多线程任务 (需要修改multi_jobs_demo.php中username和password)

$ cd tutorial
$ php multi_jobs_demo.php

3: 单线程任务

$ php single_job_demo.php
